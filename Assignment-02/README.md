# Assignment 02 — Personal Portfolio Page

## Aim
Design a personal portfolio page using HTML and CSS, with Bootstrap CDN.

## Technologies Used
- HTML
- CSS
- Bootstrap 5 (via CDN)

## Steps Performed
1. Built a responsive navbar, hero section, About, Skills, Projects, and Contact sections in `index.html`.
2. Styled the page with custom CSS in `style.css` (gradient hero, skill boxes, project cards, footer).
3. Linked GitHub project repositories and LinkedIn profile.
4. Verified responsiveness on desktop and mobile.
5. Captured output screenshots.

## Output
Screenshots are available in `output-screenshots/`.

This portfolio content was reused (with additions) to build the repository's hosted **profile-website** in `/docs`.
