# Assignment 01 — HTML Text Formatting Tags

## Aim
Create a static web page which defines all text formatting tags of HTML in tabular format.

## Technologies Used
- HTML
- CSS (internal styling)

## Steps Performed
1. Created `index.html` demonstrating heading tags, paragraph, bold/italic text, ordered and unordered lists, `div`, tables, images, and hyperlinks.
2. Styled headings and table borders using an internal `<style>` block.
3. Opened the file in a browser to verify rendering.
4. Captured output screenshots.
