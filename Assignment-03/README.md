# Assignment 03 — Mini E-Commerce Web App

## Aim
Design a dynamic web app like a fashion store / e-commerce product store using HTML, CSS & JavaScript.

## Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulation, event handling)

## Steps Performed
1. Built a product listing (`index.html`) with a navbar, hero banner, and product grid.
2. Implemented cart logic in `script.js`: add-to-cart, dynamic cart rendering, total calculation, checkout, and final bill screens.
3. Styled the storefront in `style.css` (grid layout, hover animation, responsive cards).
4. Tested the full flow: Home → Cart → Checkout → Bill.
5. Captured output screenshots.

## Output
Screenshots are available in `output-screenshots/`.
