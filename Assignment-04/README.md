# Assignment 04 — Weather Monitoring Dashboard

## Aim
Create an interactive data visualization dashboard for a real-life application (Weather Monitoring Dashboard). Visualize data using charts and graphs.

## Technologies Used
- HTML
- CSS
- D3.js (Data-Driven Documents)

## Steps Performed
1. Built a dashboard with a city-selection dropdown (Pune, Mumbai, Delhi, Nanded, Nagpur, Hyderabad, Bengaluru, Chennai, Jaipur).
2. Used D3.js to render an animated line chart for weekly temperature, humidity, and rainfall.
3. Added a bar chart for average temperature across all cities.
4. Added a bar chart for total weekly rainfall.
5. Styled the dashboard with clean, card-based UI.
6. Captured output screenshots.
